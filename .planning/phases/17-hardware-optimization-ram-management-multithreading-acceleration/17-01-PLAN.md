## Goal

Create the MemoryMonitor foundation module (`shared/memory.py`) with RAM tracking, adaptive chunk sizing, checkpoint utilities, and lifecycle cleanup helpers. Add chunk_size passthrough to CSV loaders. Extend `build_labels()` and `build_iric()` signatures with `n_jobs`/`max_ram_gb` parameters (defaults preserve existing behavior) and wire them through `run_labels()`/`run_iric()` in pipeline.py.

## must_haves

- [ ] `MemoryMonitor` class exists in `shared/memory.py` with `check()` returning status at 90%/100% thresholds against `max_ram_gb`
- [ ] `adaptive_chunk_size()` halves chunk size at 90% usage, with a minimum floor (1000 rows)
- [ ] Checkpoint save/load to Parquet implemented in `shared/memory.py`
- [ ] `cleanup()` helper for lifecycle `del` + `gc.collect()` available in `shared/memory.py`
- [ ] `_load_csv()` and public loader functions accept optional `chunk_size` parameter (override `settings.chunk_size`)
- [ ] `build_labels(force, n_jobs, max_ram_gb)` — extended signature (params default, no-op when not passed)
- [ ] `build_iric(force, n_jobs, max_ram_gb)` — extended signature (params default, no-op when not passed)
- [ ] `run_labels(cfg)` and `run_iric(cfg)` pass `n_jobs` and `max_ram_gb` from PipelineConfig
- [ ] All existing tests pass; new unit tests for MemoryMonitor, adaptive chunk sizing, and checkpoint round-trip

<task id="17-01-01" title="Create shared/memory.py — MemoryMonitor + utilities" wave="1" depends_on="" files="src/sip_engine/shared/memory.py">

Create `src/sip_engine/shared/memory.py` with the following components:

### MemoryMonitor class
```python
class MemoryMonitor:
    def __init__(self, max_ram_gb: int):
        self.budget_bytes = max_ram_gb * (1024**3)
    
    def current_usage_bytes(self) -> int:
        """Return current process RSS via psutil.Process().memory_info().rss."""
    
    def usage_ratio(self) -> float:
        """Return current_usage_bytes / budget_bytes."""
    
    def check(self) -> str:
        """Return 'ok' (<90%), 'warning' (90-100%), or 'critical' (>=100%)."""
```

### adaptive_chunk_size function
```python
def adaptive_chunk_size(monitor: MemoryMonitor, base_chunk_size: int, min_chunk_size: int = 1000) -> int:
    """Return base_chunk_size if ok, halved if warning, min_chunk_size if critical."""
```

### Checkpoint utilities
```python
def save_checkpoint(rows: list[dict], checkpoint_path: Path) -> None:
    """Write rows to a temp Parquet file, then atomically rename to checkpoint_path.
    Uses safe_rename from compat.py for Windows safety."""

def load_checkpoint(checkpoint_path: Path) -> tuple[pd.DataFrame, set]:
    """Load checkpoint Parquet. Return (df, set_of_processed_ids).
    The set is built from the 'id_contrato' column for skip-based resume."""

def remove_checkpoint(checkpoint_path: Path) -> None:
    """Delete checkpoint file if it exists."""
```

### Lifecycle cleanup helper
```python
def cleanup(*objects) -> None:
    """del each object reference and call gc.collect() once."""
```

**Note:** Use `from sip_engine.compat import safe_rename` for atomic checkpoint writes. Import `psutil`, `gc`, `pandas`, `Path` from standard/existing deps. No new dependencies.

6. **Pipeline wiring:** `run_labels` and `run_iric` pass `n_jobs` and `max_ram_gb` from `PipelineConfig`
