**Add to trainer.py** (appending to the infrastructure from Plan 07-01):

`train_model(model_id: str, force: bool = False, quick: bool = False, n_iter: int = 200, n_jobs: int = -1) -> Path`:

Complete training pipeline for one model. Steps:

1. **Validate model_id** — assert model_id in MODEL_IDS. Raise ValueError otherwise.

2. **Check force / existing artifacts** — `model_dir = settings.artifacts_models_dir / model_id`. If `model_dir / "model.pkl"` exists and not force, log "Model {model_id} already exists, use --force to retrain" and return model_dir. Create model_dir if it doesn't exist (`model_dir.mkdir(parents=True, exist_ok=True)`).

3. **Load features + labels** — Read `settings.features_path` (features.parquet) and `settings.labels_path` (labels.parquet) with pyarrow. If either file doesn't exist, raise FileNotFoundError with clear message: "Run 'python -m sip_engine build-features' first" / "Run 'python -m sip_engine build-labels' first".

4. **Merge on id_contrato** — features.parquet has id_contrato as index; labels.parquet has id_contrato column. Reset features index first (`features_df.reset_index()`), then merge via inner join on id_contrato. After merge, set id_contrato as the named index: `merged = merged.set_index('id_contrato')`. This ensures X_train.index and X_test.index carry id_contrato throughout the pipeline and into test_data.parquet. Extract X = merged[FEATURE_COLUMNS], y = merged[model_id].

5. **Drop NaN labels** — `mask = y.notna(); X = X[mask]; y = y[mask].astype(int)`. Log how many rows dropped and final dataset size. If no positive examples, log warning and return early.

6. **Stratified split** — Call `_stratified_split(X, y)`. Log train/test sizes and class distributions.

7. **IRIC threshold recalibration on train-only (IRIC-08)** — Call `calibrate_iric_thresholds(X_train_df)` where `X_train_df` is the training DataFrame (must still have the feature columns needed by the calibration function). Then call `save_iric_thresholds(train_thresholds)` to persist the train-only thresholds for future online inference.
   ```python
   from sip_engine.iric.thresholds import calibrate_iric_thresholds, save_iric_thresholds
   train_thresholds = calibrate_iric_thresholds(X_train)
   save_iric_thresholds(train_thresholds)
   ```
   Log a note: "IRIC thresholds recalibrated on train-only data. Note: iric_* feature column VALUES in features.parquet still reflect full-dataset calibration from Phase 6. The saved thresholds are for future online inference." Document this limitation in training_report.json under `"iric_note"`.

7b. **Encoding mappings recalibration on train-only** — Call `build_encoding_mappings(X_train, force=True)` to recompute and persist `encoding_mappings.json` using only training-set distributions. This ensures the "Other" categorical grouping threshold reflects train-only frequencies for future online inference. Note: the integer-encoded values already in features.parquet are NOT changed (they were pre-encoded in Phase 6); this step saves train-only mappings for the `compute_features()` online path.
   ```python
   from sip_engine.features.encoding import build_encoding_mappings
   build_encoding_mappings(X_train, force=True)
   ```
   Log: "Encoding mappings recomputed on train-only data for online inference."

8. **Quick mode adjustments** — If quick=True: set n_iter=min(n_iter, 20), n_splits=3. Otherwise n_splits=5.

9. **HP search** — Call `_hp_search(X_train.values, y_train.values, n_iter=n_iter, n_splits=n_splits, seed=RANDOM_SEED, n_jobs=n_jobs, device_kwargs=_detect_xgb_device())`. Log best strategy, best AUC-ROC, best params.

10. **Final refit** — Create XGBClassifier with best_params + device_kwargs + `objective='binary:logistic'` + `verbosity=0` + `random_state=RANDOM_SEED`. If best_strategy is "scale_pos_weight", also set `scale_pos_weight = (y_train == 0).sum() / max((y_train == 1).sum(), 1)`. If best_strategy is "upsampling_25pct", upsample minority in full training set before fitting (same logic as _cv_score_upsampling but on full train). Fit on a DataFrame with FEATURE_COLUMNS as column names so `feature_names_in_` is set automatically. **CRITICAL: This is the final model that gets saved -- NOT a CV fold model.**

11. **Save model.pkl** — `joblib.dump(clf, model_dir / "model.pkl")`.

12. **Save feature_registry.json** — Per RESEARCH.md Pattern 4 schema:
    ```python
    {
        "model_id": model_id,
        "feature_columns": FEATURE_COLUMNS,
        "n_features": len(FEATURE_COLUMNS),
        "training_date": datetime.utcnow().isoformat() + "Z",
        "label": model_id,
        "best_strategy": search_result["best_strategy"],
        "best_params": {k: (int(v) if isinstance(v, (np.integer,)) else float(v) if isinstance(v, (np.floating,)) else v) for k, v in search_result["best_params"].items()},
        "cv_auc_roc_mean": float(search_result["best_cv_auc_mean"]),
        "cv_auc_roc_std": float(search_result["best_cv_auc_std"]),
        "train_size": int(len(y_train)),
        "test_size": int(len(y_test)),
        "class_distribution": {"0": int((y_train == 0).sum()), "1": int((y_train == 1).sum())},
        "random_seed": RANDOM_SEED,
    }
    ```
    Use json.dumps with indent=2, ensure all numpy types are cast to Python natives.

13. **Save training_report.json** — Per RESEARCH.md Pattern 4 schema:
    ```python
    {
        "model_id": model_id,
        "label_distribution": {"0": int((y == 0).sum()), "1": int((y == 1).sum())},
        "scale_pos_weight_value": float((y_train == 0).sum() / max((y_train == 1).sum(), 1)),
        "strategy_comparison": {strategy comparison from best HP candidate},
        "all_cv_results": search_result["all_results"],
        "best_params": ...,
        "training_duration_seconds": float,
        "feature_count": len(FEATURE_COLUMNS),
        "seed": RANDOM_SEED,
        "n_iter": n_iter,
        "n_splits": n_splits,
        "quick_mode": quick,
        "iric_note": "IRIC thresholds recalibrated on train-only and saved for online inference. iric_* feature column VALUES in features.parquet still reflect full-dataset calibration from Phase 6.",
        "encoding_note": "encoding_mappings.json recomputed on train-only for online inference. features.parquet integer codes unchanged.",
    }
    ```
    Ensure all numpy types serialized to Python natives.

14. **Save test data** — Save X_test and y_test as `test_data.parquet` in model_dir. Construct: `test_df = X_test[FEATURE_COLUMNS].copy(); test_df[model_id] = y_test`. X_test.index is already the named `id_contrato` index (set in step 4 and preserved through `_stratified_split` which wraps `train_test_split`). Assert `test_df.index.name == 'id_contrato'` before saving. Phase 8 loads this for evaluation.

15. **Log completion** — "Model {model_id} trained in {duration:.1f}s. Saved to {model_dir}".

16. Return model_dir.

**Update models/__init__.py** — Add `train_model` to re-exports: `from sip_engine.models.trainer import train_model, MODEL_IDS`.

**Numpy serialization helper** — Create a small helper `_json_safe(obj)` that recursively converts numpy types to Python natives for JSON serialization. Or use a custom JSONEncoder subclass.

```python
                           help='Parallelism level (default: -1 = all cores)')
```

In the command dispatch block, replace the "not yet implemented" fallthrough with:

```python
elif args.command == "train":
    from sip_engine.models.trainer import train_model, MODEL_IDS
    models_to_train = [args.model] if args.model else MODEL_IDS
    try:
        for mid in models_to_train:
            model_dir = train_model(
                model_id=mid,
                force=args.force,
                quick=args.quick,
                n_iter=args.n_iter,
                n_jobs=args.n_jobs,
            )
            print(f"Model {mid} trained: {model_dir}")
        sys.exit(0)
    except FileNotFoundError as e:
        print(f"Error: {e}", file=sys.stderr)
        sys.exit(1)
    except Exception as e:
        print(f"Error training models: {e}", file=sys.stderr)
        sys.exit(1)
```
