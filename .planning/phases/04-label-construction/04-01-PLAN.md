    Functions to implement in this task:

    a) `_load_contratos_base() -> pd.DataFrame`:
       - Use `load_contratos()` generator from `sip_engine.data.loaders` to stream chunks
       - Select only needed columns: `["ID Contrato", "TipoDocProveedor", "Documento Proveedor"]`
       - Concatenate all chunks, then `drop_duplicates(subset=["ID Contrato"], keep="first")`
       - Log count of rows before/after dedup: `logger.info("Contratos loaded: %d rows (%d unique)", total, unique)`
       - Return the deduplicated DataFrame

    b) `_build_m1_m2_sets(contratos_ids: set[str]) -> tuple[set[str], set[str]]`:
       - Accept a set of valid contract IDs (from contratos) for filtering
       - Stream adiciones.csv via `load_adiciones()` generator
       - For each chunk: filter to rows where `id_contrato` is in `contratos_ids`
       - For matched rows: `tipo_upper = row["tipo"].str.strip().str.upper()`
       - Collect contract IDs into `m1_contracts: set[str]` where tipo_upper is in M1_TIPOS
       - Collect contract IDs into `m2_contracts: set[str]` where tipo_upper is in M2_TIPOS
       - Track orphan count (rows where id_contrato NOT in contratos_ids)
       - Track total rows processed for diagnostics
       - Log: `logger.info("Adiciones processed: %d total, %d matched, %d orphans (%.1f%%)", ...)`
       - Return `(m1_contracts, m2_contracts)`

    c) `build_labels(force: bool = False) -> Path` (SKELETON ONLY in this task):
       - Check if `settings.labels_path` exists and `force` is False -> return early with log
       - Check that `settings.rcac_path` exists -> raise `FileNotFoundError("RCAC index not found. Run 'python -m sip_engine build-rcac' first.")`
       - Create `settings.artifacts_labels_dir` directory (mkdir parents=True, exist_ok=True)
       - Call `_load_contratos_base()` to get contratos DataFrame
       - Build contratos_ids set from `df["ID Contrato"]`
       - Call `_build_m1_m2_sets(contratos_ids)` to get m1/m2 sets
       - Add M1 column: `df["M1"] = df["ID Contrato"].isin(m1_contracts).astype("Int8")`
       - Add M2 column: `df["M2"] = df["ID Contrato"].isin(m2_contracts).astype("Int8")`
       - Leave M3/M4 as placeholder (Plan 04-02 adds them)
       - For now, add `# M3 and M4 computed in _compute_m3_m4()` comment placeholder
       - Log M1/M2 summary: counts and percentages
       - If M2 positive count < 50: `logger.warning("M2 has only %d positive examples — model may not be trainable", m2_count)`
       - Return `settings.labels_path` (parquet write deferred to Plan 04-02)
