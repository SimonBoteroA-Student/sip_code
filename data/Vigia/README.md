# vigia_project
Este repositorio contiene el código usado para el desarrollo de la aplicación Vigia para la detección de ineficiencias e irregularidades en la contratación pública de Bogotá.
Este proyecto fue desarrollado mediante Jupyter notebooks en Pyhon 3.7, la configuración del entorno en conda se encuentra en el archivo **vigia_env.yml**, y en base a ello se desarrollo la respectiva aplicación.

Los datos crudos usados fueron extraidos en Febrero del 2021, por lo que la fuente de datos ha cambiado los formatos de las tablas consultadas.

Los notebooks con el código correspondiente, contenidos en la presente carpeta, deben ser ejecutados en el siguiente orden:

1. SECOP_II_preprocessing.ipynb
2. SECOP_II_EDA.ipynb
3. SECOP_II_IRIC_exploration.ipynb
4. SECOP_II_models.ipynb

Nota: Todo el contenido está en español.
