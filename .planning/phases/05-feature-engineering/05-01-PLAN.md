```python
The index enables O(1) as-of-date lookup for any provider's prior contract

            {"fecha_firma": date, "valor": float, "departamento": str, "m1": 0|1, "m2": 0|1},
            ...  # sorted ascending by fecha_firma
        ]
    }

    Steps:
    1. Load contratos via chunked loader — extract id_contrato, TipoDocProveedor, Documento Proveedor, Fecha de Firma, Valor del Contrato, Departamento
    2. Drop rows with null Fecha de Firma (7.2% — cannot be ordered temporally)
    3. Load labels.parquet — join M1/M2 on id_contrato
    4. Normalize provider IDs via normalize_tipo/normalize_numero
    5. Group by (tipo_norm, num_norm), sort each group ascending by fecha_firma
    6. Serialize with joblib.dump()
    """

def reset_provider_history_cache() -> None:

def lookup_provider_history(tipo_doc: str, num_doc: str, as_of_date: datetime.date, departamento: str | None = None) -> dict:
    """Look up provider history as-of a given date.

    Returns dict:
    {
        "num_contratos_previos_nacional": int,
        "num_contratos_previos_depto": int,
        "valor_total_contratos_previos_nacional": float,
        "valor_total_contratos_previos_depto": float,
        "num_sobrecostos_previos": int,
        "num_retrasos_previos": int,
    }

    All values are 0 for first-time providers. Departmental counts are 0 if departamento is None.
    Uses bisect_left for O(log n) cutoff on the sorted date list, then aggregates from the slice.
    Strictly < as_of_date (same-day excluded per FEAT-05).
    """
```

Key implementation details:
- Use `bisect.bisect_left` on the sorted dates list to find the cutoff index
- For national: sum all entries before cutoff
- For departmental: filter entries before cutoff where departamento matches
- M1/M2 label values may be `pd.NA` (nullable Int8) — treat as 0 when counting sobrecostos/retrasos
- Provider ID normalization: `normalize_tipo(tipo_doc)` and `normalize_numero(num_doc)` at lookup boundary (same pattern as rcac_lookup)
- If provider key not in index, return all-zeros dict immediately
