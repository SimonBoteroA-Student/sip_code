**`__init__.py`:** Re-export public symbols: `from sip_engine.evaluation.evaluator import evaluate_model, evaluate_all`

**`evaluator.py`:** Full evaluation module with these components:

**Constants:**
- `MODEL_IDS = ["M1", "M2", "M3", "M4"]`
- `THRESHOLDS = [round(t, 2) for t in np.arange(0.05, 1.0, 0.05)]` — 19 decision thresholds
- `K_VALUES = [100, 500, 1000]` — for MAP@k and NDCG@k

**Imports:** sklearn.metrics (roc_auc_score, roc_curve, brier_score_loss, ndcg_score, confusion_matrix, precision_score, recall_score, f1_score), joblib, pandas, numpy, json, csv, logging, pathlib, datetime.

**Public function: `map_at_k(y_true, y_scores, k)`** — Custom MAP@k implementation (public for testability):
1. Sort indices by y_scores descending: `sorted_indices = np.argsort(y_scores)[::-1]`
2. Take top-k true labels: `y_true_topk = y_true[sorted_indices][:k]`
3. At each position where label==1, compute precision = num_positives_so_far / (position + 1)
4. Return mean of those precisions, or 0.0 if no positives in top-k
5. Handle edge case: k > len(y_true) by clamping k = min(k, len(y_true))

**Private helpers:**

1. `_load_artifacts(model_id: str, models_dir: Path | None) -> tuple[Any, pd.DataFrame, dict, dict]`
   - Default models_dir: `Path("artifacts/models")`
   - Load `model = joblib.load(models_dir / model_id / "model.pkl")`
   - Load `test_df = pd.read_parquet(models_dir / model_id / "test_data.parquet")`
   - Load `training_report = json.loads((models_dir / model_id / "training_report.json").read_text())`
   - Load `feature_registry = json.loads((models_dir / model_id / "feature_registry.json").read_text())`
   - Raise FileNotFoundError with helpful message if any file missing (e.g., "Model M1 not found — run 'python -m sip_engine train --model M1' first")
   - Return (model, test_df, training_report, feature_registry)

2. `_compute_discrimination_metrics(y_true: np.ndarray, y_scores: np.ndarray) -> dict` (EVAL-01)
   - `auc = roc_auc_score(y_true, y_scores)`
   - `fpr, tpr, thresholds = roc_curve(y_true, y_scores)`
   - Return `{"auc_roc": float(auc), "roc_curve": {"fpr": fpr.tolist(), "tpr": tpr.tolist(), "thresholds": thresholds.tolist()}}`

3. `_compute_ranking_metrics(y_true: np.ndarray, y_scores: np.ndarray) -> dict` (EVAL-02, EVAL-03)
   - For each k in K_VALUES: compute `map_at_k(y_true, y_scores, k)`
   - For each k in K_VALUES: compute `ndcg_score(y_true.reshape(1, -1), y_scores.reshape(1, -1), k=k)`
   - Return `{"map_100": float, "map_500": float, "map_1000": float, "ndcg_100": float, "ndcg_500": float, "ndcg_1000": float}`

4. `_compute_calibration_metrics(y_true: np.ndarray, y_scores: np.ndarray) -> dict` (EVAL-05)
   - `brier = brier_score_loss(y_true, y_scores)`
   - `positive_rate = y_true.mean()`
   - `brier_baseline = positive_rate * (1 - positive_rate)`
   - Return `{"brier_score": float(brier), "brier_baseline": float(brier_baseline)}`

5. `_compute_threshold_analysis(y_true: np.ndarray, y_scores: np.ndarray) -> dict` (EVAL-04)
   - For each threshold in THRESHOLDS (19 total):
     - `y_pred = (y_scores >= threshold).astype(int)`
     - Compute precision, recall, f1 using sklearn (zero_division=0)
     - Compute confusion matrix: `tn, fp, fn, tp = confusion_matrix(y_true, y_pred, labels=[0, 1]).ravel()`
   - Find optimal threshold (F1-maximizing) across the sweep
   - Return dict with "thresholds", "precision", "recall", "f1", "confusion_matrices" (list of dicts), and "optimal_threshold" sub-dict with value/precision/recall/f1/confusion_matrix

6. `_get_output_path(output_dir: Path, model_id: str, extension: str) -> Path`
   - Base path: `output_dir / model_id / f"{model_id}_eval{extension}"`
   - If base path doesn't exist → return it
   - If it exists → add timestamp: `{model_id}_eval_{YYYY-MM-DD_HH-MM-SS}{extension}`
   - Ensure parent directory exists (mkdir -p)

7. `_write_json_report(eval_dict: dict, output_path: Path) -> None`
   - Write eval_dict as indented JSON (indent=2)
   - Full nested structure per RESEARCH.md schema: model_id, evaluation_date, test_set_size, label_distribution, discrimination, ranking, calibration, threshold_analysis, optimal_threshold, training_context

8. `_write_csv_report(eval_dict: dict, output_path: Path) -> None`
   - Combined format: threshold rows (one per threshold with precision/recall/f1/confusion matrix values) followed by summary rows for scalar metrics (auc_roc, brier_score, map_100, map_500, map_1000, ndcg_100, ndcg_500, ndcg_1000, optimal threshold)
   - Columns: metric_type, threshold, value, precision, recall, f1, tn, fp, fn, tp
   - Use csv.writer (stdlib) — pandas overhead unnecessary for simple tabular output

9. `_write_markdown_report(eval_dict: dict, output_path: Path) -> None`
   - Generate human-readable markdown following RESEARCH.md template structure:
     - Header with model ID, timestamp, test set size, positive rate
     - Discrimination Metrics table (AUC-ROC)
     - Ranking Metrics table (MAP@100/500/1000, NDCG@100/500/1000)
     - Calibration Metrics table (Brier Score + baseline)
     - Threshold Analysis table (all 19 thresholds with P/R/F1/confusion matrix)
     - Optimal Threshold section
     - Training Context section (best params, strategy, CV scores)
   - Reference JSON/CSV files for full data

**Public function: `evaluate_model(model_id: str, models_dir: Path | None = None, output_dir: Path | None = None) -> Path`**
   - Default output_dir: `Path("artifacts/evaluation")`
   - Step 1: Load artifacts via `_load_artifacts`
   - Step 2: Extract X_test and y_test from test_df using feature_registry column names; y_test = test_df[model_id].values
   - Step 3: Generate y_pred_proba = model.predict_proba(X_test)[:, 1]
   - Step 4: Compute all metric categories (discrimination, ranking, calibration, threshold_analysis)
   - Step 5: Build eval_dict assembling: model_id, evaluation_date (ISO 8601 UTC), test_set_size, label_distribution (counts + positive_rate), all metric dicts, training_context from training_report
   - Step 6: Log metrics to console as they're computed (verbose output per CONTEXT.md):
     ```
     Evaluating model M1...
       ✓ Loaded model and test data (N samples, X% positive)
       ✓ AUC-ROC: 0.XXXX
       ✓ Brier Score: 0.XXXX
       ✓ MAP@100: 0.XXXX, MAP@500: 0.XXXX, MAP@1000: 0.XXXX
       ✓ NDCG@100: 0.XXXX, NDCG@500: 0.XXXX, NDCG@1000: 0.XXXX
       ✓ Analyzed 19 decision thresholds (optimal: 0.XX, F1=0.XX)
       ✓ Reports written to artifacts/evaluation/M1/
     ```
   - Step 7: Write JSON, CSV, Markdown reports via _get_output_path + respective writers
   - Return output directory path

**Public function: `evaluate_all(models_dir: Path | None = None, output_dir: Path | None = None) -> Path`**
   - Call evaluate_model() for each model in MODEL_IDS
   - Collect key metrics from each model's eval_dict
   - Write summary.json (nested dict with all models' key metrics: auc_roc, brier_score, map_100/500/1000, ndcg_100/500/1000, optimal_threshold, precision_at_optimal, recall_at_optimal, test_set_size, positive_rate)
   - Write summary.csv (one row per model, columns matching JSON keys)
   - Print console summary table (use simple formatted string table — avoid adding tabulate dependency in this plan; Plan 02 handles that)
   - Return output_dir path

Use standard project logging pattern: `logger = logging.getLogger(__name__)`.
