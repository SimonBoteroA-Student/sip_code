  <name>Task 1: Create rcac_lookup module with lazy-loading O(1) lookup</name>
  <files>src/sip_engine/data/rcac_lookup.py, tests/test_rcac.py</files>
  <action>
1. Create `src/sip_engine/data/rcac_lookup.py`:

   ```python
   """O(1) RCAC lookup by (document_type, document_number).

   The RCAC index is loaded lazily on first call to rcac_lookup() and cached
   in module-level state. Subsequent calls use the in-memory dict.

   Both arguments to rcac_lookup() are normalized internally using the same
   normalization functions from rcac_builder — callers do NOT need to
   pre-normalize inputs.
   """
   ```

   **Module-level state:**
   - `_rcac_index: dict | None = None` (starts as None, loaded on first use)

   **Functions:**

   - `_load_rcac() -> dict`: Load `settings.rcac_path` via `joblib.load()`. Raise `FileNotFoundError` if pkl does not exist. Store in module-level `_rcac_index`. Return the dict.

   - `get_rcac_index() -> dict`: Return `_rcac_index` if loaded, else call `_load_rcac()`. Useful for testing/inspection.

   - `reset_rcac_cache() -> None`: Set `_rcac_index = None`. Used by tests to clear state between test runs.

   - `rcac_lookup(tipo_doc: str, num_doc: str) -> dict | None`:
     1. Normalize inputs: `tipo = normalize_tipo(tipo_doc)`, `num = normalize_numero(num_doc)`
     2. Check `is_malformed(num)` — if True, return None
     3. Get index via `get_rcac_index()`
     4. Return `index.get((tipo, num))` — None if not found

   Import normalization functions from `sip_engine.data.rcac_builder`: `normalize_tipo`, `normalize_numero`, `is_malformed`.
   Import `get_settings` from `sip_engine.config`.
   Use `logging.getLogger(__name__)` for logging.

2. Add lookup tests to `tests/test_rcac.py` (append to existing file from Plan 03-01):

   - `test_lookup_hit_returns_record`: Build RCAC with fixture data, call `rcac_lookup('CC', '12345678')`, assert returns dict with expected keys (tipo_documento, numero_documento, en_boletines, ..., num_fuentes_distintas, malformed).
   - `test_lookup_miss_returns_none`: `rcac_lookup('CC', '99999999')` -> None
   - `test_lookup_normalizes_input`: `rcac_lookup('CC', '43.922.546')` matches record stored as '43922546'
   - `test_lookup_malformed_returns_none`: `rcac_lookup('CC', '12')` -> None (short number, malformed)
   - `test_lookup_en_multas_secop_flag`: Build with multas source data, verify `en_multas_secop=True` on the returned record.
   - `test_lookup_without_pkl_raises`: With no pkl file, `get_rcac_index()` raises `FileNotFoundError`.

   Each lookup test should:
   1. Use the `rcac_source_dirs` fixture from Plan 03-01
   2. Call `build_rcac(force=True)` to create the pkl
   3. Call `reset_rcac_cache()` to clear module state
   4. Then call `rcac_lookup()` or `get_rcac_index()`

   Add an autouse fixture for rcac tests that calls `reset_rcac_cache()` in teardown:
   ```python
   @pytest.fixture(autouse=True)
   def _reset_rcac(self):
       yield
       from sip_engine.data.rcac_lookup import reset_rcac_cache
       reset_rcac_cache()
   ```
   (Or add it at module level if not using a class.)

3. Run tests: `pytest tests/test_rcac.py -v`
