**Helper: `_count_lines(path) -> int`** — runs `subprocess.run(["wc", "-l", str(path)])`, parses output, returns line count. Handles subprocess failure gracefully (returns 0 if wc fails, tqdm will just show a spinner).

**Helper: `_total_chunks(path, chunk_size) -> int`** — `(_count_lines(path) - 1 + chunk_size - 1) // chunk_size`. The -1 accounts for the header row in headed files. For headerless files, do not subtract 1.

**Helper: `_BadRowCounter(logging.Handler)`** — counts `ParserWarning` "Skipping line" messages per research Pattern 4. Used by each loader to report `rows_skipped` in the summary log.

**Core pattern for each HEADED loader (contratos, procesos, ofertas, proponentes, proveedores, boletines, ejecucion, suspensiones, adiciones, resp_fiscales, colusiones_sic, sanciones_penales):**

```python
def load_contratos() -> Generator[pd.DataFrame, None, None]:
    """Yield chunks of contratos_SECOP.csv with critical dtypes enforced and currency columns cleaned."""
    settings = get_settings()
    path = settings.contratos_path
    if not path.exists():
        raise FileNotFoundError(f"contratos not found: {path}")

    validate_columns(str(path), CONTRATOS_USECOLS, encoding="utf-8")

    total = _total_chunks(path, settings.chunk_size)
    rows_loaded = 0

    # Set up bad-row counter
    logging.captureWarnings(True)
    counter = _BadRowCounter()
    logging.getLogger("py.warnings").addHandler(counter)

    t0 = time.time()
    reader = pd.read_csv(
        path,
        chunksize=settings.chunk_size,
        usecols=CONTRATOS_USECOLS,
        dtype=CONTRATOS_DTYPE,
        encoding="utf-8",
        encoding_errors="replace",
        on_bad_lines="warn",
    )

    try:
        for chunk in tqdm.tqdm(reader, total=total, desc="contratos", unit="chunk"):
            # Clean currency columns to float
            for col in CONTRATOS_CURRENCY_COLS:
                if col in chunk.columns:
                    chunk[col] = clean_currency(chunk[col])
            rows_loaded += len(chunk)
            yield chunk
    finally:
        logging.getLogger("py.warnings").removeHandler(counter)
        elapsed = time.time() - t0
        logger.info(
            "%s: %d rows loaded, %d rows skipped, %.1fs",
            path.name, rows_loaded, counter.count, elapsed,
        )
```
